# Модификаторы доступа в C#

## 1. public
Всем везде. Любой код в любой сборке.

## 2. private
Только внутри своего класса. Никто снаружи не видит.

## 3. internal
Только внутри своего проекта (сборки). Другие проекты не видят.

## 4. protected
Внутри класса + все наследники (даже из других проектов).

## 5. protected internal
Доступно если ты наследник (protected) ИЛИ ты в своей сборке (internal).

## 6. private protected
Доступно только наследникам, и только если они в ТОЙ ЖЕ СБОРКЕ.

## 7. file (C# 11)
Только внутри своего файла. Другие файлы не видят.


## Исключения и правила

- Класс на верхнем уровне (не вложенный) может быть только: public, internal, file
- Структуры НЕ могут использовать: protected, protected internal, private protected (нет наследования)
- Вложенные классы по умолчанию: private
- Поля/методы без модификатора по умолчанию: private
